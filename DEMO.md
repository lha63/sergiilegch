# Сэргийлэгч — Демо (Web Mobile App)

Иргэдийн аюулгүй байдлын AI аппын **танилцуулга зориулалттай интерактив прототип**.
Backend шаардлагагүй — зөвхөн HTML/CSS/JS. Бүх өгөгдөл жишээ (mock).

> Бүрэн бүтээгдэхүүний шаардлагыг [PRD.md](PRD.md)-ээс үзнэ үү.

## Ажиллуулах

**Хамгийн хялбар:** `index.html`-ийг браузераар нээх (давхар товших).

**Эсвэл локал сервер** (газрын зураг найдвартай ачаалахын тулд зөвлөмжтэй):

```bash
cd e:/mobile-app
python -m http.server 5500
# дараа нь нээх: http://127.0.0.1:5500/index.html
```

> Газрын зураг (Leaflet/OpenStreetMap) ачаалахад интернэт шаардлагатай.

## Туршиж үзэх интерактив

- **SOS** товч → 3-2-1 тоолж дуудлага илгээх → туслахууд хариу өгөх
- **Watch** → "Аялал эхлүүлэх" → тоолуур ажиллаж "Аюулгүй хүрлээ"
- **Voice** → микрофон дээр дарвал ~2.6 сек дараа "SOS" таниж дохио өгнө (демо)
- **Маршрут** → 3 сонголтыг солиход газрын зураг дээрх зам өөрчлөгдөнө
- **Зураг** → эрсдэлийн бүсүүд (улаан/шар/ногоон) бодит УБ зураг дээр
- **Гэр бүлийн тойрог** → гэр бүлийн гишүүд газрын зураг дээр амьд, батерей/байршил/check-in
- **Нууц SOS** → "Хуурамч дуудлага" → дуудлагын дэлгэц → хариулбал чимээгүй дохио + бичлэг
- **Хөвөгч нууц SOS товч (ягаан, баруун доод)** → аль ч горим (маршрут, хяна, SOS...) дээр шууд → Чимээгүй SOS / Хуурамч дуудлага / Сирена
- **Уналт илрүүлэх** → "Туршиж үзэх" → 10 сек countdown → автомат SOS+бичлэг
- **Сирена** → дарвал бүтэн дэлгэц анивчиж дохиолол өгнө
- **Эрсдэлийн зураг** → Өдөр/Шөнө сонголтоор эрсдэлийн бүс өөрчлөгдөнө
- **Иргэдийн анхааруулга** → "Мэдээлэл өгөх" → төрөл сонгох sheet

## Аппын файлууд

```
index.html   — бүрхүүл: утасны хүрээ, SVG icon sprite, доод навигац
styles.css   — design token + бүх компонент (dark theme)
data.js      — жишээ өгөгдөл (УБ дүүрэг, эрсдэл, guardian, камер)
app.js       — router, дэлгэц бүрийн render, Leaflet газрын зураг, интерактив
PRD.md       — бүтээгдэхүүний шаардлагын баримт
```

> Тэмдэглэл: `README.md`, `CLAUDE.md`, `.claude/`, `src/`, `cli/` зэрэг нь
> `ui-ux-pro-max` дизайн skill суулгахад орж ирсэн файлууд — аппад хамаарахгүй.

## Дизайн систем

`ui-ux-pro-max` skill-ийн зөвлөмжөөр:
- **Загвар:** "Accessible & Ethical" (төрийн/нийтийн үйлчилгээ, WCAG нийцтэй)
- **Өнгө:** Emergency SOS & Safety (улаан `#DC2626` + цэнхэр `#2563EB`) + шөнийн navy суурь
- **Фонт:** Inter
- **A11y:** SVG icon (emoji биш), focus ring, ≥44px touch target, `prefers-reduced-motion`, өнгөнөөс гадна icon/label

## Хязгаарлалт (демо тул)

- Бодит backend, нэвтрэлт, мэдээллийн сан байхгүй
- Дуу хоолой таних, байршил, камер бүгд дуурайлган харуулсан
- Газрын зураг гадаад tile (CartoDB/OSM) ашигладаг тул офлайн үед хэсэгчлэн харагдана
