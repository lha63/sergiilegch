# 🛡️ Сэргийлэгч — Иргэдийн аюулгүй байдлын AI апп (Демо)

AI-д суурилсан, иргэдийн хувийн болон гэр бүлийн аюулгүй байдлыг бодит цагт хамгаалах гар утасны аппын **танилцуулга прототип**.

> Backend шаардлагагүй — зөвхөн HTML/CSS/JS. Бүх өгөгдөл жишээ (mock). Газрын зураг: Leaflet + OpenStreetMap.

## ✨ Боломжууд (14 горим)

**Яаралтай үед:** Нууц SOS + бичлэг · Уналт/осол илрүүлэх · Эмнэлгийн SOS карт · Сирена
**Урьдчилан сэргийлэх:** AI эрсдэлийн зураг · Аюулгүй маршрут · Хорооны оноо · Аюулгүй цэгүүд · Иргэдийн анхааруулга
**Хяналт ба хамтын ажиллагаа:** Намайг гэртээ хүртэл хяна · Гэр бүлийн тойрог · Community Guardian · Дуу хоолойгоор SOS · AI камерын анализ

➕ Аль ч горим дээр **хөвөгч нууц SOS товч**-оор шууд тусламж дуудна.

## 🔴 Live горим (бодит цагийн backend — MVP)

Профайл → **"Live горим"** дотор Firebase-д суурилсан **бодит цагийн** функцууд ажиллана:
- **Амьд байршил** — хүүхдийн байршил ахлагчийн зураг дээр real-time хөдөлнө
- **Шуурхай SOS** — хүүхэд SOS дарвал ахлагчид дуу+чичиргээ+байршилтай шууд анхааруулга
- **Гэртээ хүртэл хяна** — аяллын төлөв (замдаа/аюулгүй хүрсэн) хуваалцана
- **Pairing** — ахлагч 4 оронтой код үүсгэж, хүүхэд кодоор холбогдоно

> ⚙️ Ажиллуулахын тулд үнэгүй Firebase project хэрэгтэй — [LIVE-SETUP.md](LIVE-SETUP.md) (5-10 мин).
> Тохиргоо хийгээгүй ч бусад 14 горим mock-оор бүрэн ажиллана.

## 🚀 Локалаар ажиллуулах

`index.html`-ийг браузераар нээх, эсвэл локал сервер:

```bash
python -m http.server 5500
# http://127.0.0.1:5500
```

## 🌐 GitHub Pages дээр үнэгүй deploy хийх

1. GitHub дээр шинэ repo үүсгэх (жишээ нэр: `sergiilegch`).
2. Энэ фолдерын файлуудыг push хийх:
   ```bash
   git init
   git add .
   git commit -m "Сэргийлэгч демо"
   git branch -M main
   git remote add origin https://github.com/<хэрэглэгч>/sergiilegch.git
   git push -u origin main
   ```
3. GitHub repo → **Settings → Pages** → *Build and deployment* → Source: **Deploy from a branch** → Branch: **main** / **/(root)** → Save.
4. 1-2 минутын дараа сайт нээгдэнэ:
   `https://<хэрэглэгч>.github.io/sergiilegch/`

> Газрын зураг, фонт нь интернэтээс ачаалагдах тул deploy хийсэн сайт онлайн ажиллана.
